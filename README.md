# Smart Notes App

A browser-based Smart Notes application that allows users to create, organize, and manage notes efficiently.  
The base layout and structure were generated using VibeCoding, with additional smart and interactive features implemented manually.

## âœ¨ Core Features

- Create, edit, and delete notes
- Notes persist using browser `localStorage`
- Clean, responsive card-based layout
- Light and dark mode with saved user preference
- Pin important notes to keep them at the top
- Sort notes by newest or oldest
- Real-time, case-insensitive search
- Export all notes as `.json` or `.txt`
