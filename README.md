# Smart Notes App

A lightweight Smart Notes single-page app with add/edit/delete, localStorage persistence, dark mode, searching, pinning, sorting, and simple smart metadata (summary/tags/mood). Built using VibeCoding scaffold & extended manually.

## Features
- Add / Edit / Delete notes
- Notes persisted in browser `localStorage`
- Pin important notes (pinned notes stay on top)
- Sort notes by newest / oldest
- Real-time, case-insensitive search with highlighted matches
- Light & Dark mode (preference saved)
- Mock AI-generated summary, tags, and mood on note creation
- Export notes as `.json` or `.txt` (download)
- Optional: basic Markdown rendering

## How to run (development)
1. `git clone <repo>`
2. `cd smart-notes`
3. Open `index.html` (or run `npm start` if a dev server is included)
